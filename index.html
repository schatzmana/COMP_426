Final Project Title: Carolina "Craigslist"

Description of your final project:

Our project is going to be a craigslist-type website, where users can log on to sell and buy items. There will be a home page, from which you can then go to a Create a Post page, which will be a form where you can input what you are selling, category, price, and a description. Users will be required to log in before they can create a post. Users can then go onto a page where they will be able to view all of their past posts and delete their posts. We will also have a Results page where everyone can see every single post that has been made. We hope to be able to sort these results based on category, price, and other attributes. Our posts, usernames, and categories will all be stored in a relational database that we will retrieve all of our data from.
